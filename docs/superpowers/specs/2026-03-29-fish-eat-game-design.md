# 鱼吃鱼游戏 — 设计文档

## 1. 项目概述

- **项目名称：** 鱼吃鱼
- **类型：** 单机俯视角吃鱼 PK 游戏
- **核心玩法：** 玩家操控鱼，通过吃更小的鱼和电脑对战来升级、获得装备和技能
- **目标用户：** 休闲游戏玩家

## 3. 核心玩法

### 3.1 战斗机制

- **体型规则：** 鱼越大能吃越小的鱼，被更大的鱼吃
- **血量系统：** 每条鱼有 HP，被攻击减少，HP=0 时死亡
- **速度差异：** 不同鱼速度不同，影响追逐和逃跑
- **特殊克制：** 不同鱼种有属性相克（石头剪刀布式）
  - 例：鲨鱼 > 小鱼 > 虾 > 鲨鱼
- **技能影响：** 带技能的鱼 vs 普通鱼，各有优劣
- **胜负条件：** 一方 HP 归零或 PK 时间耗尽

### 3.2 操作控制

| 按键 | 功能 |
|------|------|
| 方向键 ↑↓←→ | 移动 |
| Shift | 加速（消耗体力） |
| Q/W/E/R | 释放技能（1-4 槽位） |

## 4. 成长体系

### 4.1 等级系统

- **经验获取：**
  - 吃鱼：获得经验值（根据被吃鱼的大小）
  - 连续击杀：短时间内连续吃鱼，额外经验奖励
  - PK 胜利：每场 PK 胜利获得大量经验

- **升级效果：**
  - 提升基础属性（血量、速度）
  - 解锁新技能（1级解锁技能1，5级解锁技能2...）

### 4.2 技能系统

- **技能数量：** 最多 4 个主动技能槽位
- **解锁方式：** 随等级解锁
- **技能分支：**
  - 攻击系：伤害类技能
  - 防御系：护盾/回血类技能
  - 辅助系：加速/隐身类技能

### 4.3 道具获取

- **宝箱掉落：** 击杀敌人后概率掉落
- **宝箱内容：**
  - 金币
  - 药水（血量恢复、加速等）
  - 随机技能

## 5. 漂流瓶系统

### 5.1 触发机制

- 升级时自动触发
- 触发概率受幸运值影响

### 5.2 效果列表

**🟢 好的效果（概率高）：**
| 效果 | 持续时间/次数 |
|------|--------------|
| 双倍金币 | 1 分钟 |
| 血量全满 | 即时 |
| 技能冷却加速 | 30 秒 |
| 随机技能宝箱 | 即时 |
| 金币 +50 | 即时 |
| 金币 +100 | 即时 |
| 金币 +200 | 即时 |
| 幸运值 +1 | 整局 |
| 幸运值 +2 | 整局 |

**🔴 坏的效果（概率低）：**
| 效果 | 持续时间/次数 |
|------|--------------|
| 移动速度 -30% | 10 秒 |
| 视野缩小 | 10 秒 |
| 技能冷却 +50% | 10 秒 |
| 金币 -30 | 即时 |
| 幸运值 -1 | 整局 |

### 5.3 幸运值机制

- **基础幸运值：** 初始为 0
- **影响范围：**
  - 漂流瓶触发概率
  - 漂流瓶效果好坏概率
- **提升方式：** 皮肤/装备永久加成（后续版本）

## 6. 难度与关卡

### 6.1 难度分级

| 难度 | 解锁条件 | 敌人数量 | AI 强度 | 敌人属性 |
|------|---------|---------|---------|---------|
| 简单 | 默认 | 3-5 | 低 | 血量低、速度慢 |
| 普通 | 通关简单 | 5-8 | 中 | 血量中、速度中 |
| 困难 | 达到特定等级 | 8-12 | 高 | 血量高、速度快、技能多 |

### 6.2 关卡设计

- **分房间制：** 每场 PK 一个独立房间
- **胜利条件：** 击败所有敌人 / 坚持到时间结束
- **地图主题切换：**
  - 关卡 1-3：深海主题（珊瑚礁、海藻）
  - 关卡 4-6：热带浅海主题（阳光、海草）
  - 关卡 7-9：极地主题（冰山、热泉）
  - 关卡 10+：循环，难度递增

### 6.3 地图环境

**深海主题：**
- 深蓝渐变海水
- 光线从上方透入
- 障碍物：珊瑚礁、沉船残骸
- 气泡漂浮

**热带浅海主题：**
- 明亮蓝绿海水
- 阳光直射效果
- 障碍物：海草、礁石
- 热带鱼群游动

**极地/深海主题：**
- 深蓝到黑色渐变
- 障碍物：冰山碎片、海底火山热泉
- 极光效果

## 7. 视觉设计

### 7.1 美术风格

- **主风格：** 深海氛围（深蓝海水、神秘感）
- **点缀：** 卡通元素（鲜艳色彩、圆润造型）
- **整体感觉：** 有氛围感又不失趣味

### 7.2 HUD 布局

采用 **简约 HUD** 方案：

```
┌─────────────────────────────────────┐
│ ❤️ HP ████████  💰 1280  ⭐ Lv.5   │  ← 顶部状态栏
│                                     │
│                                     │
│           游戏区域                   │
│                                     │
│                                     │
│                     ┌────┐          │
│                     │地图│          │
│  [Q] [W] [E🔴] [R] └────┘          │  ← 底部技能栏
└─────────────────────────────────────┘
```

- **顶部：** 血量条、金币、等级
- **底部中央：** 技能栏（Q/W/E/R），冷却中显示红色
- **右下角：** 小地图
