# Product Description -- 鱼吃鱼 (Fish Eat Fish)

## What Is This?

鱼吃鱼是一款单人的 Phaser.js 3.x HTML5 游戏，玩家控制一条鱼吃更小的鱼、与 AI 敌人战斗、升级、解锁技能。

## Core Features

### 游戏机制

- **吃鱼规则** — 玩家必须比目标大 20% 以上才能吃掉
- **属性克制** — 剪刀石头布式（鲨鱼 > 小鱼 > 虾 > 鲨鱼）
- **升级** — 提升体型、解锁技能、触发漂流瓶效果
- **难度** — 简单/普通/困难，敌人数量和 AI 侵略性递增
- **玩家体型** — 玩家为基础体型的 2 倍（便于识别）

### 操控方式

| 输入 | 动作 |
|-----|------|
| 方向键 ↑↓←→ | 移动 |
| Shift | 加速 |
| Q | 撕咬技能（伤害） |
| W | 护盾技能（无敌） |
| E | 加速技能（速度提升） |
| R | 治疗技能（回血） |
| 鼠标 | 替代移动方式（点击移动） |

### 特殊鱼行为（5种）

| 鱼类型 | 特殊行为 | 效果 |
|--------|---------|------|
| 鳗鱼 (Eel) | 冲刺攻击 | 短距离高速冲刺造成伤害 |
| 章鱼 (Octopus) | 隐身 | 短暂隐身，无法被敌人检测 |
| 海马 (Seahorse) | 逃跑 | HP 低时主动逃离玩家 |
| 水母 (Jellyfish) | AOE | 周期性释放范围伤害 |
| 灯笼鱼 (Anglerfish) | 远程攻击 | 发射投射物攻击玩家 |

### 波次系统

游戏使用动态波次系统：
- **Calm** — 平静期，刷怪间隔 8 秒
- **Surge** — 高峰期，刷怪间隔 4 秒
- **Peak** — 峰值，刷怪间隔 2 秒

波次状态循环：calm → surge → peak → calm

### 技能系统

| 技能 | 快捷键 | 冷却 | 效果 |
|------|-------|------|------|
| 撕咬 | Q | 5s | 对目标造成伤害 |
| 护盾 | W | 15s | 3秒无敌，免疫伤害 |
| 加速 | E | 12s | 移动速度 +50%，持续 5s |
| 治疗 | R | 20s | 恢复 30% HP |

### 成就系统

15 个里程碑成就：
- 吃掉 10/50/100 条鱼
- 达到等级 5/10/15/20
- 获得 1000/5000/10000 分数
- 生存 1/5/10 分钟
- 击败 5/10/20 鲨鱼
- 其他特殊挑战

### 漂流瓶系统

升级时有概率触发漂流瓶效果（好/坏随机）：
- **好的效果** — 双倍经验、护盾、回血
- **坏的效果** — 减速、虚弱、召唤敌人

### 商店系统

游戏内商店可购买升级：
- 永久增加 HP 上限
- 永久增加伤害
- 永久增加移动速度
- 永久增加经验获取

### 数据持久化

- **认证** — localStorage 认证会话
- **用户数据** — localStorage 存档（分数、等级、设置）
- **会话** — 跨页面刷新保持登录状态

## 约束

- **最大体型比例** — 玩家只能吃比自己小 20% 的鱼
- **最小支持分辨率** — 800x600
- **浏览器要求** — WebGL 支持
- **最大敌人数量** — 根据难度动态调整

## 世界坐标与深度系统（ScrollingWorld）

### 世界尺寸

- **尺寸**: 20000 × 20000 像素
- **出生点**: (10000, 14000) — 深海区域，向上可探索至浅海

### 深度区域（5 个）

玩家沿 worldY 轴移动时，背景颜色和装饰类型随深度变化：

| 区域 ID | worldY 范围 | 名称 | 颜色基调 | 主要装饰 |
|---------|------------|------|---------|---------|
| `surface` | 0 ~ 2000 | 浅海 | 亮青绿 | 阳光光柱、小气泡 |
| `shallow` | 2000 ~ 4000 | 珊瑚礁 | 青蓝 | 珊瑚群 |
| `mid` | 4000 ~ 8000 | 中层 | 深蓝 | 海草、礁石 |
| `deep` | 8000 ~ 12000 | 深海 | 暗蓝 | 大岩石、暗礁 |
| `abyss` | 12000 ~ 20000 | 深渊 | 接近黑 | 发光水母轮廓 |

### 深度雾效

- worldY < 12000: 无雾
- worldY = 12000 ~ 20000: 雾 alpha 从 0 线性增至 0.65
- 雾色: `#010818`（深蓝黑）

### 视差层（3 层）

| 层 | scrollFactorX | 视觉效果 |
|----|--------------|---------|
| 背景层 | 0.08 | 最慢，几乎静止 |
| 中层 | 0.25 | 中速 |
| 前景层 | 0.50 | 快速，与玩家速度接近 |

### 方向语义

- **向上 (worldY 减少)**: 从深海进入浅海，颜色变亮，装饰从发光水母变为阳光光柱
- **向下 (worldY 增加)**: 从浅海进入深海，颜色变暗，装饰从珊瑚变为发光水母
- **向左/向右**: 视差效果产生卷轴感，装饰类型由 chunk 种子决定（同一位置可复现）

## 性能目标

- 游戏加载: <3 秒
- 帧率: 60 FPS
- 碰撞检测: O(n) 优化
- 敌人 AI: 每帧 <1ms
