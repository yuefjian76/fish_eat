## 💡 关键概念速览

### 方案1: 视觉层次 (简单)
```
问题: 敌人、粒子、UI混在一起，视觉噪点
解决: 创建深度标准 
  背景(1-5) → 敌人(50) → 玩家(100) → UI(100-210) → 模态(300)
工作量: 1天
效果: 视觉清晰度⬆️30%
```

### 方案2: 打击反馈 (中等)
```
问题: 伤害反馈无层次，感觉不爽快
解决: 按伤害等级分类反馈
  10伤害 → 轻微震+黄数字
  25伤害 → 明显震+红数字
  50伤害 → 强烈震+亮红大字
工作量: 3天
效果: 爽快感⬆️40%, 满足感⬆️50%
```

### 方案3: 克制系统 (中等)
```
问题: fish.json有克制数据但没激活
解决: 
  1. 伤害计算加克制乘数 (克制×1.8, 被克×0.6)
  2. HUD显示最近敌人的克制状态
工作量: 2天
效果: 策略意识⬆️50%, 重玩变化⬆️40%
```
