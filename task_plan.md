### A1: 刷怪权重系统
**问题**: 当前spawn完全等概率，低级玩家会遇到鳗鱼/灯笼鱼等高难度敌人
**修复**: 按玩家等级动态调整各鱼出现概率
```javascript
// 实现于 GameScene._getSpawnWeights()
level <= 3: { clownfish: 0.4, shrimp: 0.35, shark: 0.15, jellyfish: 0.1 }
level <= 6: { clownfish: 0.2, shrimp: 0.2, shark: 0.2, jellyfish: 0.15, seahorse: 0.15, octopus: 0.1 }
level <= 10: { 小丑鱼: 0.1, 小虾: 0.1, 鲨鱼: 0.15, 灯笼鱼: 0.15, 水母: 0.1, 海马: 0.15, 章鱼: 0.15, 鳗鱼: 0.1 }
level > 10: { 鲨鱼: 0.2, 灯笼鱼: 0.2, 水母: 0.15, 海马: 0.1, 章鱼: 0.15, 鳗鱼: 0.2 }
```

### A2: 大小对比光晕提示
**问题**: 玩家不知道哪条鱼可以吃
**修复**: 根据相对大小显示绿边（可吃）或红边（危险）
```javascript
// 可吃：玩家size > 鱼size * 1.2 → 绿边
// 危险：鱼size > 玩家size * 1.2 → 红边
```

### A3: 受伤击退效果
**问题**: 被攻击时只有屏幕震动，缺少"被打退"感知
**修复**: 被攻击时给玩家短暂反向速度脉冲

### A4: 悬浮数字全覆盖
**问题**: 只有bite技能有伤害数字，碰撞伤害/吃鱼无反馈
**修复**: 吃鱼显示+EXP，被打显示-HP
