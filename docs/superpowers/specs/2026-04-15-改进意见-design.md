# Fish Eat Fish - 改进方案设计文档

## 1. 视觉反馈增强

### 1.1 命中闪光效果
- **目标**：增强玩家吃鱼时的反馈
- **方案**：
  - 添加短暂的白色光晕（0.2秒）覆盖鱼身
  - 使用 `this.tweens.addScale` 实现缩放动画
  - 颜色使用主题色系（金色/橙色）

### 1.2 死亡动画
- **目标**：提升被吃掉的视觉体验
- **方案**：
  - 执行缩放动画（0.8→1.2→0）
  - 添加粒子爆发（使用 `createEatParticles` 类似逻辑）
  - 同步音效播放

### 1.3 宝箱开启效果
- **目标**：提升奖励收集的仪式感
- **方案**：
  - 旋转动画 + 粒子效果
  - 延迟0.3秒后显示内容
  - 解锁时播放特殊音效

## 2. 游戏手感提升

### 2.1 移动平滑化
- **当前问题**：瞬间加速/停止生硬
- **方案**：
  ```javascript
  // 添加加速度
  const ACCELERATION = 800;
  const MAX_SPEED = 200;
  this.player.setAcceleration(ACCELERATION);
  ```

### 2.2 变速系统
- 根据鱼的大小设置最大速度
- 小鱼更快但控制难度高

### 2.3 连击系统
- 连续吃鱼时显示连击数字
- 连击越高得分加成越大

## 3. 无障碍选项

### 3.1 色盲友好模式

### 3.2 控制选项
