## 📊 核心问题诊断

### 当前游戏的痛点

| 维度 | 问题 | 影响 |
|------|------|------|
| 🎮 **打击感** | 伤害反馈离散化、无层次感 | 玩家感不到"爽快" |
| 🧠 **策略深度** | 技能选择无差异、克制形同虚设 | 所有局游都一样 |
| 🎵 **节奏感** | 敌人生成机械化、无韵律感 | 感觉无聊单调 |
| 🏆 **留存感** | 无成就、无里程碑、无变化性 | 长期留存差 |

### 根本原因

> 游戏**看似完整**（8种敌鱼、4个技能、3个Boss），但各系统**各自为政**，缺乏**贯通性设计**和**反馈层次感**。

## 🔑 关键改进详解

### 方案1: 视觉层次重构 ⭐ 快速赢利

**问题**: 敌人、粒子、UI 深度混乱，视觉噪点大

**解决方案**: 创建统一的深度标准
```
背景(1-5) < 敌人(50) < 玩家(100) < 浮动文本(150) < HUD(99-210) < 模态(300)
```

**预期效果**: 视觉清晰度 ⬆️30%

**工时**: 1天 | **难度**: 简单

---

### 方案2: 打击反馈系统 ⭐ 核心改进

**问题**: 伤害反馈无层次，玩家感不到打击感

**解决方案**: 按伤害等级分层反馈
```
MINOR(0-10)    → 轻微震 + 黄色数字
NORMAL(10-25)  → 明显震 + 红色数字
HEAVY(25-50)   → 强烈震 + 深红数字
CRITICAL(50+)  → 极致震 + 亮红大字 + 远距飘浮
```

**预期效果**: 爽快感 ⬆️40%, 满足感 ⬆️50%

**工时**: 3天 | **难度**: 中等

---

### 方案3: 克制系统强化 ⭐ 策略激活

**问题**: fish.json 中有克制数据但未被激活

**解决方案**: 
- 增加伤害计算中的克制乘数 (克制 ×1.8, 被克 ×0.6)
- HUD 显示最近敌人的克制指示

**预期效果**: 策略选择意识 ⬆️50%, 重玩变化性 ⬆️40%

**工时**: 2天 | **难度**: 中等

---

### 方案4: 节奏感系统 (新)

**问题**: 敌人以固定间隔(2秒)生成，节奏机械化

**解决方案**: 
- 计算威胁度 = 敌人数量 + 玩家血量比
- 动态调整生成频率 (4秒→1秒)
- 波次系统: 每击杀5条鱼推送一次

**预期效果**: 节奏沉浸感 ⬆️40%

**工时**: 2天 | **难度**: 中等

---

### 方案5: 技能多样化

**问题**: 4个技能都是独立作用，无协同

**解决方案**:
- 新增3个技能（冲刺、反制、连击刺）
- 技能间有协同关系（×1.3-1.5伤害倍数）
- 创建 `SkillSynergySystem`

**预期效果**: 重玩价值 ⬆️60%, 战术深度 ⬆️50%

**工时**: 4天 | **难度**: 复杂

---

### 方案6: 里程碑系统

**问题**: 无中期目标感，玩家不知道要做什么

**解决方案**:
```json
"first_kill": 击杀第一条鱼
"level_5_boss": 达到5级挑战Boss
"five_combo": 达成5连击
```

**预期效果**: 目标感 ⬆️60%, 参与度 ⬆️40%

**工时**: 1.5天 | **难度**: 简单
